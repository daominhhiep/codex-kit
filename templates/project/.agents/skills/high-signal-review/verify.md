# High-Signal Review Verification

Before closing the review, verify that:

1. The scope was named clearly.
2. Findings are ordered by severity.
3. Each finding explains:
   - what is wrong
   - why it matters
   - where it is
   - how it can be observed or triggered
4. Style-only feedback did not crowd out material issues.
5. If there are no findings, residual risks or testing gaps are still called out.
